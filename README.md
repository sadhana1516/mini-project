Student Management System (SMS) with Database & Reporting
This is a practical application I built to streamline how student data is handled. It replaces old-school manual entry with a structured database and adds a reporting feature to save time on administrative tasks.

👉  Why I Built This (Problem Statement)
Managing student records in small schools or coaching centers is often a mess. Most places either use physical registers or basic Excel sheets which lead to:

Data Loss: Files get deleted by accident or registers get damaged.

Wasted Time: It takes way too long to find a specific student's record from six months ago.

Manual Reporting: Generating monthly performance or attendance reports usually takes hours of manual work.

My Solution: This system provides a clean, searchable interface. All data is stored in a secure SQL database, and I’ve added a one-click feature to generate professional reports instantly.

👉  Key Features
Full CRUD Support: Add new students, view profiles, update details, or delete records.

Instant Search: Find any student record by searching their Name or ID.

Automated Reports: Generate printable PDF/Excel reports for grades and attendance.

Database Reliability: Uses SQL to ensure data stays organized and safe.

Simple Interface: Focuses on usability so anyone can operate it without a manual.

🛠️ Tech Stack
Language:  Python 

Database:  MySQL

UI Framework:  Tkinter 

Reporting:  Pandas

⚙️ How to Setup & Run
To get this project running on your local machine:

Clone the repository:

Bash
git clone https://github.com/sadhana1516/student-management-system.git
Setup the Database:

Import the database_setup.sql file (found in the /db folder) into your SQL server.

Install Dependencies:

Run this command to install required libraries:

Bash
pip install -r requirements.txt
Launch:

Run the main application file:

Bash
python main.py
